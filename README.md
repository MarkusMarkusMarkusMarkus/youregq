# YourEDQ - Egregoric Diversity Quotient

Measure your fluency across collective belief systems.

Live at: [youregq.com](https://youregq.com)

## What is this?

A quiz that measures your "egregoric diversity quotient" - how many different collective belief systems, subcultures, and memetic environments you can fluently navigate.

## Deploy to GitHub Pages

This repo is configured to automatically deploy to GitHub Pages.

1. Push code to GitHub
2. Enable GitHub Pages in Settings → Pages → Source: GitHub Actions
3. The site will auto-deploy on every push to main
4. Add custom domain in Settings → Pages if desired
